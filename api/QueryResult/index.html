QueryResult

An iterable object returned as the result of [Route:query] that can filter snapshots by Identifier and Senders.

:::note Due to certain limitations with the Luau Type System, iterating over the QueryResult Object will not return typed values. In order to fix this, call :__iter() on the QueryResult Object.

For example:

for i, sender, ... in net:query():__iter() do
	-- Do something
end

:::

See Querying Data for more information.

Usage:
